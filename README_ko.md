# Sleekes (슬리케스)

**Sleekes**는 `yt-dlp`를 코어로 사용하는 범용 동영상 아카이빙 시스템이자 솔루션입니다. 단순한 다운로드를 넘어, 영상의 모든 메타데이터와 자산을 영구히 보존하기 위해 설계되었습니다.

## 초월적 기능 업데이트 (Sleekes Pro)

- **오디오 단독 추출**: 인공지능이 최상의 음질(MP3)로 음원만 따로 보관합니다.
- **안티 밴(Anti-Ban) 시스템**: 유튜브의 IP 차단을 막기 위해 사람처럼 랜덤하게 휴식하는 지능형 대기 모드를 탑재했습니다.
- **플랫폼 확장**: 유튜브를 넘어 인스타그램(Reels), 틱톡(워터마크 제거), 페이스북 등 글로벌 플랫폼을 완벽 지원합니다.
- **쿠키 연동**: 브라우저(Chrome, Edge 등)의 쿠키를 빌려와 로그인해야 볼 수 있는 비공개 영상이나 연령 제한 영상도 아카이빙합니다.
- **플레이리스트 정밀 제어**: 재생목록 전체 혹은 특정 범위(예: 1~10번 영상)만 골라 받는 기능이 추가되었습니다.
- **데이터 아카이빙**: 영상 없이 설명, 댓글, 메타데이터만 초고속으로 수집하는 모드를 지원합니다.

## 설치 및 사용법

### 가상환경 활성화
```bash
source venv/bin/activate
```

### GUI 실행
```bash
python main.py
```

### CLI 실행
```bash
python main.py [URL] [옵션]
# 예: 모든 메타데이터와 함께 아카이빙 (5초~10초 랜덤 휴식 적용)
python main.py https://youtube.com/watch?v=... --archive --sleep 5 --max-sleep 10
```

## CLI 주요 옵션 (신규)
- `-x`, `--audio`: 오디오만 MP3로 추출
- `--skip-video`: 영상 제외, 데이터(댓글/설명 등)만 수집
- `--sleep [초]`: 영상 사이의 휴식 시간 설정
- `--cookies [브라우저]`: 쿠키를 가져올 브라우저 선택 (chrome, edge 등)
- `--playlist-items [범위]`: 플레이리스트 다운로드 범위 (예: 1-5,10)
- `--flat`: 복잡한 폴더 구조 없이 현재 폴더에 파일 저장
