# Sleekes

**Sleekes** is a universal video archiving system and solution powered by `yt-dlp`. It is designed to archive not just videos, but every piece of metadata and asset associated with them.

## Key Features
- **Universal Archiving**: Collect video, description, metadata JSON, subtitles, thumbnails, and comments.
- **Dual Mode Support**: Modern GUI (PySide6) and powerful CLI.
- **Powered by yt-dlp**: Seamless support for thousands of video platforms.
- **Premium Design**: Focused on aesthetics and user experience.

## Quick Start
1. Activate virtual environment: `source venv/bin/activate`
2. Run GUI: `python main.py`
3. Run CLI: `python main.py [URL] [OPTIONS]`
